# CLAUDE.md

## Project

Personal portfolio website built with plain HTML and CSS — no frameworks, no build tools.

## Style

- Dark theme throughout
- Modern and clean aesthetic: generous whitespace, clear typography, minimal decoration
- CSS custom properties for colors and spacing so the palette stays consistent

## Git

- Descriptive commit messages that explain the intent, not just the change (e.g., "add project cards section to portfolio" not "update index.html")
- Small, focused commits — one logical change per commit

## Conventions

- No JavaScript unless strictly necessary
- No external CSS frameworks or libraries
- Keep HTML semantic (`<section>`, `<article>`, `<nav>`, etc.)
- Prefer relative units (`rem`, `%`, `vw`) over fixed pixels
